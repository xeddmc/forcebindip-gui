# forcebindip-gui
Java Based GUI for ForceBindIP


## Installation

Make sure you have the JRE installed ( http://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html )

Then run the .jar

## Usage

Double click the .jar file
